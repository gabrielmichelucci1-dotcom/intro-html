Atividade Classroom: Arquivo HTML

O que é um arquivo HTML? E quais suas Utilidades

O Arquivo HTML é um arquivo de texto criado por Tim Berners-Lee e é utilizado na internet até nos dias de hoje, esse arquivo contém código HTML, ou seja, é um arquivo que pode ser aberto em um navegador web para exibir uma página web. Ele é usado para criar a estrutura e o conteúdo de uma página web, como texto, imagens, links, etc. O HTML é a linguagem de marcação usada para criar páginas web e é essencial para o desenvolvimento de sites e aplicações web.

AS SUAS PRINCIPAIS UTILIDADES SÃO:

  • Criar estrutura de páginas web
  • Adicionar conteúdo e elementos visuais
  • Estabelecer links entre diferentes páginas

ESTRUTURA DE UM ARQUIVO HTML:

Um arquivo HTML geralmente tem a seguinte estrutura básica:

  • DOCTYPE: Define o tipo de documento e a versão do HTML
  • ELEMENTO BODY: Contém o conteúdo visível da página web
  • ELEMENTO HEAD:Contém informações sobre o documento, como o título e meta tags
  • ELEMENTO HTML: Envolve todo o conteúdo do documento e define a linguagem
  • ELEMENTO TITLE: Define o título da página que aparece na aba do navegador
  • ELEMENTO LANG: Define o idioma da página que aparece na aba do navegador

    • Agora vamos descobrir um pouco mais sobre os códigos e tags que contem o HTML:

    Mas primeiro vamos saber oque são as Tags:

    TAG: É um elemento de marcação usado para definir a estrutura e o conteúdo de uma página web. As tags são usadas para criar elementos como parágrafos, títulos, links, imagens, etc. Elas são escritas entre colchetes angulares (<>) e geralmente vêm em pares, com uma tag de abertura e uma tag de fechamento.

    EXEMPLO DE TAGS:

    • TAG DE ABERTURA: Precisa manter os colchetes angulares com aletra da tag dentro

    • TAG DE FECHAMENTO: Precisa ter os mesmos colchetes angulares com a tecla barra / antes da letra

    OUTRAS TAGS COMUNS:

    • TAG DE PARÁGRAFO:

      O famoso "p", usada para criar um parágrafo de texto

    • TAG DE TÍTULO: H1, H2, H3, H4, H5, H6 (AVISO: NÃO EXISTEM MAIS DE 6 TAGS DE TÍTULO, pois seria desnecessário). Usada para criar títulos e subtítulos
    • TAG DE LINK: Usada para criar um link para outra página ou recurso
    • TAG DE IMAGEM: Usada para inserir uma imagem na página

    ELEMENTOS ESPECÍFICOS E SUAS UTILIDADES NO SITE/WEB:

  • ELEMENTO META: Usado para fornecer informações sobre a página, como palavras-chave, descrição e autor

  • ELEMENTO STRONG: Usado para dar destaque a um texto, tornando-o em negrito

  • ELEMENTO EM: Usado para enfatizar um texto, tornando-o em itálico

  • ELEMENTO MARK: Usado para destacar um texto, geralmente com uma cor de fundo amarela

  • ELEMENTO SUB: Usado para criar um texto subscrito, ou seja, um texto menor que fica abaixo da linha do texto normal

  • ELEMENTO SUP: Usado para criar um texto sobrescrito, ou seja, um texto menor que fica acima da linha do texto normal

  • ELEMENTO CITE: Usado para indicar a fonte de uma citação, ou seja, o nome do autor ou da obra citada

  • ELEMENTO Q: Usado para indicar uma citação curta, ou seja, uma citação que pode ser incluída dentro de um parágrafo

  • ELEMENTO HR: Usado para criar uma linha horizontal, ou seja, serve para separar conteúdos

  • ELEMENTO B: Usado para dar destaque a um texto, tornando-o em negrito

  • ELEMENTO I: Usado para enfatizar um texto, tornando-o em itálico

  • ELEMENTO BLOCKQUOTE: Usado para indicar uma citação longa, ou seja, uma citação que ocupa um bloco separado do texto normal

    • CURIOSIADES E CITAÇÕES:

    CURIOSIDADE: O HTML é uma linguagem de marcação que evoluiu ao longo dos anos, com várias versões lançadas desde a sua criação em 1991. A versão mais recente é o HTML5, que foi lançado em 2014 e é amplamente utilizado atualmente.

    CITAÇÃO ESPECIAL DO CRIADOR TIM BERNERS-LEE:

    HTML é a linguagem de marcação que usamos para criar páginas web. Ele é a base da web e é essencial para o desenvolvimento de sites e aplicações web". -Tim Berners-Lee

    CONCLUSÃO:

    Em conclusão, o arquivo HTML é um componente essencial da web, permitindo que os desenvolvedores criem páginas e aplicações web de forma fácil e acessível. Ele é a base para a construção de sites e aplicações web, e continuará a ser uma parte fundamental da internet por muitos anos.
