conhecendo o elemento de Imagens (IMG)

sobre o elemento IMG

trabalhar com imagens no HTML envolve muito mais do que apenas colocar uma foto na página

existem várias formas, tecnicas e recursos para exibir, organizar, otimizar e adaptar imagens para diferentes dispositivos.

formatos mais comuns de imagens

  • JPEG (Joint Photographic Experts Group): é um formato de imagem amplamente utilizado para fotografias e imagens com muitos detalhes. Ele utiliza compressão com perda, o que significa que a qualidade da imagem pode ser reduzida para diminuir o tamanho do arquivo.
  • PNG (Portable Network Graphics): é um formato de imagem sem perda, ideal para gráficos, logotipos e imagens com transparência. Ele suporta uma ampla gama de cores e é amplamente utilizado na web.
  • GIF (Graphics Interchange Format): é um formato de imagem que suporta animações. Ele é limitado a 256 cores, o que o torna mais adequado para gráficos simples e animações curtas.
  • SVG (Scalable Vector Graphics): é um formato de imagem vetorial que pode ser redimensionado sem perda de qualidade. Ele é ideal para logotipos, ícones e gráficos que precisam ser escalados para diferentes tamanhos.

dimensões em pixels

você pode controlar o tamanho da imagem usando os atributos width e height. Esses atributos definem a largura e a altura da imagem em pixels

width é o atributo que define a largura da imagem em pixels.

height é o atributo que define a altura da imagem em pixels.

trabalhando com imagens responsivas

O picture é um elemento que permite oferecer diferentes versões de uma imagem para diferentes contextos, como tamanhos de tela ou resoluções.

essa tag permite:

  • Oferecer imagens otimizadas para diferentes dispositivos, como smartphones, tablets e desktops.
  • Melhorar o desempenho do site, carregando apenas a imagem necessária para o dispositivo do usuário.
  • Fornecer uma melhor experiência de usuário, garantindo que as imagens sejam exibidas corretamente em diferentes tamanhos de tela.

    • srcset- define as diferentes versões da imagem para diferentes resoluções.
