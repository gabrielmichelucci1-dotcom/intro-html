conhecendo o elemento de Iframes

Os iframes são elementos HTML que permitem incorporar outro documento HTML dentro de uma página web.

O elemento de iframe é representado pela tag <iframe> e é amplamente utilizado para exibir conteúdo de outras fontes, como vídeos, mapas, formulários ou até mesmo páginas inteiras dentro de uma página web.
