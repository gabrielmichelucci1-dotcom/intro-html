conhecendo links

sobre o elemento(A)

o elemento A é um dos elementos mais fundamentais e amplamente utilizados no HTML, pois é responsável por criar links que permitem a navegação entre páginas da web, recursos e outros destinos na internet.

no html, a gente ulitilza os links para:

  • criar links para outras páginas da web
  • acessar recursos externos
  • navegar entre seções de uma mesma página
  • iniciar ações específicas, como enviar um email ou fazer uma ligação telefônica

    • conhecendo o elemento a

    O elemento usado para criar links é a tag (anchoer)

    • href - hypertext reference
    • target - especifica onde o link será aberto
    • _blank - abre o link em uma nova aba ou janela
    • _self - abre o link na mesma aba ou janela
    • _parent - abre o link na aba pai
    • _top - abre o link na janela inteira
