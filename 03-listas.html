Conhecendo as Listas

Listas são elementos importantes em HTML para organizar informações de forma hierárquica semântica

Ordered List (lista ordenada)

Uma lista ordenada é uma lista em que os itens são numerados ou marcados com letras para indicar a ordem ou a hierarquia dos elementos.

Lista de compras

  1. Maçã
  2. Banana
  3. Leite

As listas são compostas pelo elemento LI.

Unordered List (lista não ordenada)

Uma lista não ordenada é uma lista em que os itens são marcados com símbolos, como pontos ou quadrados, sem indicar uma ordem específica.

Lista de tarefas

  • Limpar a casa
  • Fazer compras
  • Estudar para a prova

CURIOSIDADE IMPORTANTE

Você pode colocar listas dentro de listas

    Uma lista de descrição é uma lista em que cada item é composto por um termo e uma descrição associada a esse termo.
